Collage Generator

2020
p5.js

This project combines collage, a very tactile and hands-on practice, with screen based technology. Users can input a word and the program will generate a collage based on the input. This work explores how the artist's hand can be conveyed through a screen based work. Each collage element was collected and hand torn. Then each piece was scanned in to become digital assets for use in the code.
