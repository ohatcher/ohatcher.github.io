This 3D printed project is about escapism and finding small moments of joy in stressful times. The object itself is meant to provide a soothing visual escape, where the user can turn the dial to slowly filter through different colored lights. The sculpture contains references to TV and nature, which are both elements that I personally tend to go to for an escape.
